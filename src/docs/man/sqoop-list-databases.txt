sqoop-list-databases(1)
=======================

NAME
----
sqoop-list-databases - List databases available on a server.

SYNOPSIS
--------
'sqoop-list-databases' <generic-options> <tool-options>

'sqoop list-databases' <generic-options> <tool-options>


DESCRIPTION
-----------

List databases present on a server.

OPTIONS
-------

The +--connect+ option is required.

include::common-args.txt[]

COMPATIBILITY
-------------

This operation is only compatible with HSQLDB, MySQL and Oracle. When using
with Oracle, it is necessary that the user connecting to the database has DBA
privileges.

ENVIRONMENT
-----------

See 'sqoop(1)'
